== Description ==

It's very simple and handy plugin for rating a post via AJAX. It displays a star automatically right below the post title. It provides a simple 
admin panel under settings menu as well.


== Installation ==

In order to install this tiny plugin you have to follow the simple 2 steps. They are as follows:

1. Upload or copy the folder `ajax-5-star-rating-for-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Hurray! You are done ! 
Star Rating panel will be displayed in your post detail page.
And admin menu for selecting star color and managing rating data is also provided in admin panel.

enjoy!!!
